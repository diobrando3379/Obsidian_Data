## WSL

### WSL开启虚拟机

```
bcdedit /set hypervisorlaunchtype off
```

```
bcdedit /set hypervisorlaunchtype on
```

on是启用WSL，off是关闭

### vscode免密码登录

cd 到 .ssh文件夹下 (如果当前目录下没有 .ssh文件夹，就自己创建一个)

执行命令(**fd90**是**密钥串命名**)：

```
ssh-keygen -t rsa -f fd90
```

执行命令：

```
ssh-copy-id -i fd90.pub zjnu@10.158.205.107
```
